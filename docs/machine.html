🤖 Análisis de Machine Learning: Predicción de Demanda

En este proyecto, hemos desarrollado un modelo de regresión lineal para predecir la demanda de taxis en diferentes zonas de Nueva York, optimizando la asignación de recursos y mejorando la eficiencia del servicio.

Descripción de los Datos

  • ID de Ubicación de Recogida (PULocationID): Identifica zonas específicas de recogida
  • Día de la Semana: Números del 0 (Domingo) al 6 (Sábado)
  • Hora de Recogida: Formato 24 horas para análisis temporal

Metodología

Preparación de Datos

Limpieza y normalización de datos, división en conjuntos de entrenamiento y prueba.

Entrenamiento del Modelo

Implementación de regresión lineal usando Scikit-learn con características seleccionadas.

Evaluación

Uso de métricas como MSE para medir la precisión de las predicciones.

Resultados y Beneficios

Eficiencia Operacional: Optimización en la asignación de vehículos

Reducción de Costos: Minimización de tiempos muertos

Mejora del Servicio: Mayor satisfacción del cliente
